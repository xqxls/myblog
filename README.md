**项目说明**：

> 这是一个单体博客项目，博客功能主要包括登录、注册，侧边栏本周热议，用户中心、我的消息，基本设置，发表博客、评论以及消息即时通知，搜索引擎开发，网络群聊等等。

**技术选型**：

 - 核心框架：Springboot 2.1.2
 - 安全框架：Apache Shiro 1.4
 - 持久层框架：Mybatis + mybatis plus 3.2.0
 - 页面模板：Freemarker
 - 缓存框架：Redis
 - 数据库：mysql
 - 消息队列：RabbitMq
 - 分布式搜索：Elasticsearch 6.4.3
 - 双工通讯协议：websocket
 - 网络通讯框架：t-io 3.2.5
 - 工具集合：hutool 4.1.17
